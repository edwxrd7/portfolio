# Edward Henriquez — Personal Portfolio

A personal portfolio website showcasing my coursework, projects, and resume —
self-hosted from a Raspberry Pi at home rather than a cloud provider.

> Note: since this runs on home hardware, it isn't guaranteed to be publicly
> reachable 24/7. Reach out if a link is down.

## About

I'm a Computer Science senior at UNC Charlotte (Cybersecurity concentration),
graduating December 2026. This site is where I showcase my resume, hobby
projects, and a bit about myself to future employers.

Built on top of the **Dimension** template by [HTML5 UP](https://html5up.net),
then customized with a coded HTML/CSS resume section, a project showcase page,
a custom image gallery with lightbox, and a handful of other tweaks.

## Features

- Hand-coded resume section (not just an embedded PDF) with a download option
- Project showcase page for my BMW E36 325is build, including an image gallery
- Self-hosted on a Raspberry Pi at home
- Built and customized from scratch on top of an open-source template

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Font Awesome (icons)
- [Dimension](https://html5up.net/dimension) template by HTML5 UP as a base

## Running Locally

```bash
git clone https://github.com/edwxrd7/portfolio.git
cd portfolio
```

Then either open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 8000
```

and visit `http://localhost:8000`.
